# ModDownloader
Простой графический загрузчик модов для Minecraft, работающий на основе файла `mods.json`, экспортированного из **Prism Launcher**. Позволяет быстро скачать нужные моды с **Modrinth** и **CurseForge** одной кнопкой, без лишних действий.
Т.е. для создателя модпака достаточно распространить список модов(mods.json) и ссылку на эту программу.
## 🚀 Возможности

- 📥 Скачивание модов с **Modrinth** и **CurseForge**
- 📄 Чтение списка модов из `mods.json` (экспорт Prism Launcher)
- 🎮 Поддержка разных версий Minecraft (1.12.2 – 1.21.1) и популярных загрузчиков: **NeoForge, Forge, Fabric, Quilt** (в будущем добавлю ещё версий)
- 🔑 Опциональная поддержка CurseForge API‑ключа (без ключа моды с CurseForge пропускаются)
- 🧩 Автоматический поиск нужной версии мода по номеру версии и по имени файла
- ⏹️ Мгновенная остановка загрузки в любой момент
- 📋 Копирование и экспорт лога
- 💻 Полностью портативный EXE‑файл (не требует установки Python)
- 🧹 Минималистичный и понятный интерфейс

## 📥 Как использовать

1. **Получите `mods.json`**  
   В Prism Launcher:  
   Открыть сборку → Моды → Экспортировать список → JSON → И выбрать галочки: Версия, URL-адрес, Имя файла.
   Сохраните файл в удобное место.

2. **Запустите программу**  
   Скачайте последнюю версию `ModDownloader.exe` из [Releases](https://github.com/All4G/ModDownloader/releases) и откройте её.

3. **Настройте параметры**  
   - Выберите путь к `mods.json`.  
   - Укажите версию Minecraft и загрузчик (как в вашей сборке).  
   - При необходимости вставьте **CurseForge API‑ключ** (см. раздел «Получение API‑ключа»).  
   - Укажите папку, куда сохранять моды.

4. **Нажмите «Начать загрузку»**  
   Программа последовательно скачает все моды. Лог будет отображаться в реальном времени.
   
## Сборка EXE:

  pyinstaller --onefile --windowed --name "ModDownloader" mod_downloader_gui.py
Готовый файл появится в папке dist.

## Получение CurseForge API‑ключа
Зайдите на console.curseforge.com и авторизуйтесь.

Перейдите в My API Keys (или создайте новое приложение/ключ, если необходимо).

Скопируйте ключ и вставьте его в программу.

Без ключа загрузка модов с CurseForge будет невозможна, но моды с Modrinth скачиваются всегда.

## Проблемы с сетью и блокировками
Если вы используете инструменты обхода DPI (например, zapret, GoodbyeDPI), добавьте ModDownloader.exe в список исключений.
Или выключите на время скачивания.

## Формат mods.json
Программа ожидает стандартный JSON, создаваемый Prism Launcher. Пример записи:
[
  {
    "name": "3d-Skin-Layers",
    "url": "https://modrinth.com/mod/3d-skin-layers",
    "version": "1.11.2",
    "filename": "skinlayers3d-neoforge-1.11.2-mc1.21.1.jar"
  },
  {
    "name": "Advanced Netherite",
    "url": "https://curseforge.com/minecraft/mc-mods/advanced-netherite",
    "version": "1.20.1-1.2.0"
  }
]
